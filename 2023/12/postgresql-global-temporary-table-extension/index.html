PostgreSQL 全局临时表 - pgtt 问题

发表于 2023-12-22 分类于 数据库 本文字数： 18k 阅读时长 ≈ 16 分钟

PostgresQL 数据库目前不支持全局临时表，当在迁移 Oracle 数据库时，经常会遇到全局临时表的问题，因此，基本上都会借助 pgtt 来解决这个问题。最近在折腾这个插件时发现了一些问题，本文对这些问题进行了整理。

这个问题确实是由于我使用不当遇到的。当我在使用 LOAD 'pgtt'; 加载插件时，由于没有在数据库上执行 CREATE EXTENSION pgtt;，当第二次及其之后执行就会遇到这个问题，如下所示：
